PIXPY

PHOTO SHARING APP INSPIRED BY IMGUR

ABOUT THE PROJECT

Pixpy is built with a Django backend and a React frontend with a Redux architecture. Pixpy features the ability to browse and and upload photos and photo albums. The one-page web app incorporates Redux Thunk middleware to enable asynchronous actions for a cleaner frontend API. It leverages JSON Web Tokens for user authentication and data protection, and integrates the Django REST framework to implement web APIs as class mix-ins.

Software: Django, Python, JavaScript, React.js, Redux, HTML5, CSS3

Developed in collaboration with Matt Corley
